# Figma Make / Bolt Prompt — Martin Heßmann (Single Source)

## Purpose
Create a one-page personal website that feels like an authored Swiss/Bauhaus editorial piece with a soft, atmospheric layer. It must be bold, systematic, and human. Use the exact layout, copy, color, and illustration prompts below.

## Global Design System

### Typography
- Primary sans: TeX Gyre Heros (rational/structural text)
- Secondary serif: TeX Gyre Termes (emotional/intent text)
- Rule: sans for WHAT/HOW, serif for WHY/meaning
- Titles are huge and short; microcopy is precise and tight

### Color System (exact)
- Base: #FFFFFF, #F7F7F7
- Ink: #1A1A1A
- Reverse text: #FAFAFA on #1A1A1A
- Accents only: #1D1DFF, #F5D72E, #DBFF6F, #DBBEED

### Color Principles
- Accents target 10–20% of visual weight
- Gradients allowed: hero band + selected section anchors + select collage elements
- Max 2–3 accents per gradient, soft low-opacity glow, never neon
- Avoid accents in long body text

### Layout Rules
- 12-column grid, strong margins, baseline rhythm
- Grid is structural, but allow controlled breakouts/overlaps for collage moments
- One poster-scale focal element every 1–2 sections
- Optional left index rail with section numbers 01–10

### Interaction Notes (annotate)
- Principle slider changes gradient emphasis + swaps pinline illustration
- Timeline pins reveal labels on scroll
- Collage elements use subtle parallax (2–4px)
